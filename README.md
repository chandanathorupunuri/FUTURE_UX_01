# GlowBeauty Salon – Website Redesign (Task 1)

## Project Overview
This project focuses on redesigning a local beauty salon website with an emphasis on lead generation and conversion optimization.

## Screens Designed
- Homepage
- Services Page
- Contact / Lead Page

## Design Rationale
The goal of this design was to create a conversion-focused website for a local beauty salon that clearly communicates services and encourages users to book appointments. The homepage features a strong value proposition with prominent “Book Appointment” and “Call Now” CTAs to drive immediate user action. Services are presented with clear pricing to build transparency and trust. The services page organizes offerings into visually distinct cards for easy scanning and comparison. The contact page is designed as a lead capture section, including a booking form and trust elements such as ratings and appointment confirmation to reduce user hesitation. A soft color palette and beauty-focused imagery were used to match the salon brand and create a calm, premium user experience. Overall, the design prioritizes clarity, usability, and lead generation for a real-world local business.
